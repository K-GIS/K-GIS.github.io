# K-GIS.github.io

Webpage for KGIS, made the mkdocs: https://www.mkdocs.org/

install: 

    pip install mkdocs
    pip install mkdocs-bootstrap4
    pip install pymdown-extensions

Compile: 

    cd src 
    mkdocs build

Then copy everythin in folder src/site to *root*, Overwrite existing files.
